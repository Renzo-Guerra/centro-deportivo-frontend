# Centro deportivo front-end
Sistema el cual permita al dueño de un centro deportivo gestionar tanto sus canchas como los turnos que irá programando para sus clientes.

## Levantar el proyecto
* Abrir docker en tu pc y dejarlo en segundo plano
* En consola ir al root de la carpeta del proyecto
* Ejecutar el comando "docker compose up -d"
* En el navegador, ir a http://localhost:5173/ (Verificar que el puerto en el archivo docker-compose.yml tambien sea 5173)
* Cuando se quiera frenar la aplicacion, ejecutar el comando "docker compose down"
